# Installation

To install the `omics_tools` package, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/omics_tools.git
    ```

2. Navigate to the repository directory:
    ```sh
    cd omics_tools
    ```

3. Install the package using `pip`:
    ```sh
    pip install .
    ```
